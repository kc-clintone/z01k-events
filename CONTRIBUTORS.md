# Contributors

Thanks to everyone who helps improve z01k-events.

This file is intended to recognize contributors who have made meaningful improvements to the project, whether through code, documentation, design, testing, or community support.

## How to appear here

If you have contributed and would like to be listed, please open a pull request that updates this file with:

- your name or GitHub handle
- a short note about your contribution
- optionally, a link to your profile or website

Example:

- rick sanchez (@rick-sanchez) — improved the event registration flow and helped with documentation
